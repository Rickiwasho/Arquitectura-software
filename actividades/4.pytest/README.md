# Pytest para test unitarios

Los test unitarios son los encargados de probar funcionalidades específicas en nuestro código.

## Instalación
´´´sh
    $ pip install pytest 
´´´

## Cómo correr pytest?

Dirigete a la carpeta contenedora de los archivos que deseas testear, luego:

 ´´´s
